\## 🎯 Sobre o Projeto



Este livro integra \*\*Mecânica dos Fluidos\*\*, \*\*Transferência de Calor e Massa\*\*, e \*\*Modelagem Computacional\*\* em uma jornada progressiva que parte dos balanços fundamentais, avança para soluções analíticas clássicas e culmina em simulações numéricas modernas.



\### ✨ Diferenciais

\- 🎓 \*\*Duas trilhas\*\*: Graduação (conceitual) e Pós-Graduação (avançado)

\- 💻 \*\*Códigos reprodutíveis\*\*: Todos os exemplos em Python com Jupyter Notebooks

\- 🌐 \*\*QR Codes por capítulo\*\*: Acesso rápido a materiais complementares e chat com IA

\- 🔄 \*\*Problemas inversos\*\*: Calibração de parâmetros com dados experimentais

\- 🌍 \*\*Casos reais\*\*: Rio Macaé, sistemas de bombeamento, dissipadores térmicos

\## 🚀 Instalação e Uso



\### Pré-requisitos

\- Python 3.9+

\- pip ou conda



\### Instalação via pip

```bash

\\\\# Clone o repositório

git clone https://github.com/JaderLugon/fenomenos-transporte-livro.git

cd fenomenos-transporte-livro



\\\\# Crie ambiente virtual (opcional mas recomendado)

python -m venv venv

source venv/bin/activate  # Linux/Mac

\\\\# ou

venv\\\\\\\\Scripts\\\\\\\\activate  # Windows



\\\\# Instale dependências

pip install -r requirements.txt
